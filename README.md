#  Global Layoffs — Data Cleaning & EDA (Post-2020 Pandemic)

> 🚧 **Status: In Progress** — Data Cleaning & EDA complete, 
> Visualizations coming soon.

## 📌 Project Overview
This project explores layoffs carried out by companies worldwide following 
the COVID-19 pandemic (2020 onwards). Using MySQL, the raw dataset was 
cleaned and then analyzed through Exploratory Data Analysis (EDA) to 
uncover trends and insights.

## 📂 Dataset
- **Source:** Global layoffs data collected from publicly reported company layoffs
- **Time Period:** 2020 – Present
- **Fields include:** Company, Industry, Country, Total Laid Off, 
  Percentage Laid Off, Date, Funds Raised, Stage

## 🧹 Data Cleaning Steps
- Removed duplicate records
- Handled NULL and blank values
- Standardized company, industry, and country name formats
- Fixed and formatted date columns
- Removed irrelevant or unusable rows

- ## 🔍 Exploratory Data Analysis (EDA)
- Companies with the highest total layoffs
- Industries most affected by layoffs
- Countries with the most layoffs
- Year-wise and month-wise layoff trends
- Companies that laid off 100% of their workforce
- Rolling total of layoffs over time

- ## 💡 Key Insights
- The **tech industry** saw the highest number of layoffs post-pandemic
- **2022 and 2023** had the most significant layoff spikes
- **United States** had the highest number of affected companies
- Several startups laid off their **entire workforce**
